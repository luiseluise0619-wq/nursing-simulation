## 4. 치명적 버그 (수정 권고 — High)

### 4.1 고칼륨혈증 문제: 정답을 누르면 오답 처리됨
- 위치: `script.js:144` `generateElectrolyteQuestion`
- 문제 의도: "절대 **금기**인 행동을 고르시오" → 정답은 `KCL IV push` (`effect.rep: -50`).
- 판정 로직: `handleQuizChoice` 는 `choice.effect.rep > 0` 일 때 정답으로 간주 (`script.js:310`).
- **결과**: 모든 보기의 `rep`가 음수이므로 **무엇을 눌러도 오답** 처리되고, 승급 심사 모드(`generateDynamicQuestion`)에서는 `answerIdx`가 `-1` → `0` 으로 폴백되어 무작위 보기를 정답으로 표시 (`script.js:351-352`).
- **수정안**: 각 문제 객체에 명시적 `correctIndex` 또는 `isCorrect: true` 플래그를 두고, 판정 로직을 이 플래그 기준으로 바꿔야 함.

### 4.2 무균술 문제: 동일한 부호 트랩
- 위치: `script.js:156` `generateAsepticQuestion`
- 정답("시야에서 벗어난 멸균 물품도 멸균으로 간주") `rep: 20`, 나머지 `rep: -10` — 다행히 동작하지만 §4.1 과 동일 패턴이라 향후 유지보수 시 같은 함정 재발 위험. 구조적 수정 필요.

### 4.3 화상 면적 문제: `bsa` 변수 사용 안 됨
- 위치: `script.js:153` `generateBurnQuestion`
- `const bsa = pick([18, 27, 36, 45])` 로 변수를 만들지만 **이후 코드에서 한 번도 참조 안 함**. 설명문과 정답이 항상 "몸통 앞면 + 오른쪽 다리 = 36%" 로 고정.
- **수정안**: `bsa`로 분기해 설명과 정답을 동기화하거나, 변수 삭제 후 고정 문항임을 명시.

### 4.4 아프가 점수 문제: `score` 변수 사용 안 됨
- 위치: `script.js:152` `generateApgarQuestion`
- 설명에 "예시 상황 가정"이라 적혀 있으나 실제 5가지 항목(각 1점)이 본문에 박혀 있어 답은 항상 5점. `score = pick([4..9])` 가 무의미.
- **수정안**: 항목별 점수를 랜덤 생성해 본문에 삽입하고, 합을 정답으로 동적 계산.

### 4.5 네겔법 EDD 계산 경계 오류
- 위치: `script.js:151` `generateNaegeleQuestion`
- `eddDay = d + 7` 만 사용. `d` 가 최대 20 이므로 27까지만 나와 표면적으로는 문제 없지만, 윤년/2월/30일 미만 달을 무시. 또한 오답 보기 `d - 7` 이 음수일 때 폴백을 `d + 14` 로 두는 등 임시방편적.
- **수정안**: `Date` 객체로 LMP에 280일을 더해 EDD를 계산하거나, 월별 일수 테이블을 사용.

- `script.js:35-46` 유틸 함수들이 모듈로 분리되면 단위 테스트 용이.

- 접근성: 버튼만 사용하나 `aria-live` 영역 부재(로그/피드백). 스크린리더 사용자 배제.
