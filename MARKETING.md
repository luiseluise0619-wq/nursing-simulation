## 🎬 영상 컨셉 5가지 (제작 우선순위 순)

### ⭐ 1. "이 문제 풀어봐" 챌린지 (최우선)

**왜 1순위?**: 댓글 참여 유도 → 알고리즘 점수 ↑ → 노출 ↑

**구성**:
```
0~2초: 강한 후킹
"🚨 국시 1주 전, 이 1문제 못 풀면 떨어집니다"

3~15초: 문제 노출 (화면 + TTS)
"환자 BP 70/40, HR 130, SpO2 88%
다음 중 우선 중재는?
  A) Bp 강하제 IV
  B) IV 두 줄 + N/S bolus
  C) 단순 관찰
  D) 의사 콜만"

16~22초: 카운트다운 ⏱ + 정답 발표 ("B!")

23~30초: CTA
"맞췄으면 ❤️
틀렸으면 댓글로 '재도전'
앱 다운로드는 프로필 링크 👆"
```

**예상 조회수**: 2,000~50,000 (첫 1개월)

---

### ⭐ 2. 시뮬 게임플레이 모음 (정주행 1분)

**왜?**: 저장률 ↑, 알고리즘 신호 가장 강력

**구성**:
```
0~3초: 시뮬레이션 화면 빠르게 패닝
"이게 진짜 간호사 시뮬레이터입니다"

3~30초: 가장 극적인 시뮬 클립 모음
- VFib 발생 → 제세동
- ECG STEMI 인식 → 심도자실 콜
- Apgar 점수표 등장
- 자간증 경련 → MgSO4

31~50초: 결과 화면
"하루 듀티 끝, HP 60/100"

51~60초: CTA
"국시 D-X, 너도 듀티 돌래?"
```

---

### ⭐ 3. "Day vs Night 시프트" 비교 영상

**왜?**: 간호학생들이 공감하는 밈 포맷

**구성**:
```
0~2초: "간호사 시뮬 Day 시프트 vs Night 시프트"

3~15초: Day 시프트 클립
"환자 안정, 회진, 차트, 평화"

16~28초: Night 시프트 클립 
"CODE BLUE x3, 응급 입원, 자해 환자"
BGM: 긴장감 ↑

29~30초: 정적 컷
"국시 너도 듀티 돌래?"
```

---

### ⭐ 4. "국시 D-Day 카운트다운" 시리즈 (매일)

**왜?**: 시청자가 다음 영상을 기다림 → 알림 설정 ↑

**구성** (각 30초):
```
0~3초: "국시 D-XX, 오늘의 1문제"

3~25초: 일일챌린지 문제 1개 + 정답

26~30초: 
"내일 또 만나요 ✋
앱은 프로필 링크 👆"
```

**3월/9월 국시 시즌에 폭발**.

---

### ⭐ 5. ASMR / 차분한 공부 영상 (60초)

**왜?**: 야간 공부 BGM 찾는 학생들이 저장·재생

**구성**:
```
0~5초: 폰 화면 + 키보드 소리
"오늘은 모성간호학 30문제"

6~50초: 화이트 노이즈 + 앱 화면 녹화
(별다른 편집 없이 그냥 보여주기)

51~60초: 
"이 BGM 처럼 매일 공부 → 앱 링크"
```

**저장률 ↑ → 알고리즘 보너스**

## 📅 게시 일정 (3개월)

### 1개월차: 베이스 셋업
- 주 3회 게시 (월·수·금)
- 영상 컨셉 #1, #2, #3 우선 제작
- 매일 인스타 스토리 1개 ("오늘 풀이 인증" 등)
- 첫 100 팔로워 — 친구·동기·간호학과 페이지 댓글

### 2개월차: 확장
- 주 4~5회 게시
- 컨셉 #4 (카운트다운) 시작
- 인플루언서 DM 시작 (현직간호사 인플루언서 30명에게 무료 제공)
- 첫 후기 캡처 → 새 영상 소재

### 3개월차: 폭발
- 매일 게시 (단, 번아웃 X — 사전 제작 7개)
- 컨셉 #5 (ASMR) 추가 — 저장률 게임
- 광고 ₩30만 집행 (목표: CPI < ₩2,000)
- 첫 인플루언서 협찬 (Micro 5만~30만)

## 🎨 영상 제작 표준

### 사양
```
비율: 9:16 (1080×1920)
프레임: 30fps
음성: 한글 음성 OR 무음 + 자막
자막: 흰 글자 + 검은 외곽선, 폰트 16~24pt
음악: 트렌딩 오디오 (Reels에서 검색)
길이: 15~30초 (메인), 60초 (정주행용)
```

### 도구 (전부 무료)
- **CapCut** (메인 편집)
- **OBS Studio** (앱 화면 녹화)
- **Canva** (썸네일·인포그래픽)
- **Pretendard 폰트** (한글)
- **DaVinci Resolve** (고급 편집 옵션)

### 후킹 공식 (첫 2초)
1. 강한 의문문: "이 문제 풀 수 있어?"
2. 위협/긴급: "🚨 국시 1주 전 못 풀면..."
3. 호기심: "간호사들이 매일 하는 듀티가 이거다"
4. 통계: "간호학생 87%가 틀리는 문제"
5. 비교: "Day vs Night 시프트"

응답률 1~5% (3~15명 응답 예상)

티스토리 / 워드프레스 / 네이버 블로그 동시 발행.

## 💰 유료 광고 (1.5개월차부터)

### 인스타 광고 ₩30~50만/월
- 타겟: 19~25세 여성, 한국, 관심사 "nursing"
- 형식: Reels 인스타 광고 (피드+릴스 동시 노출)
- A/B 테스트: 영상 컨셉 #1 vs #3
- 예상 CPI (Cost Per Install): ₩1,500~₩3,000
- 1개월 30만원 → 100~200 다운로드

### 인플루언서 협찬 ₩10~30만/건
- Micro 인플루언서 (팔로워 1만~5만) 우선
- 협찬 내용: 30초 영상 + 스토리 1개
- 베스트 케이스 ROAS: 5x (협찬비 10만 → 50만 매출 ROI)
