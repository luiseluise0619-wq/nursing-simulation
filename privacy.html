개인정보 처리방침

간호사 시뮬레이터 (베타) · 최종 갱신: 2026-05-26 · 버전 1.1

요약: 본 앱은 학습 데이터를 사용자 기기 안에만 저장합니다. 외부 서버로 개인정보를 전송하지 않으며, 회원가입·로그인이 없습니다.

1. 수집·이용하는 정보

다음 항목만 사용자 기기의 브라우저 localStorage 에 저장됩니다. 외부 서버 전송은 없습니다.

  • 학습 통계 (과목별 정답률·콤보 최고점·세트 기록)
  • 오답 노트 (틀린 문제 + Leitner 간격 반복 메타데이터)
  • 북마크 (즐겨찾기한 문제)
  • 에피소드 진행 상태 (이어하기용) + 커리어 기록
  • 사운드·테마 설정
  • 약관 동의 일시
  • 오류 신고 로컬 기록

2. 보유·이용 기간

모든 데이터는 사용자가 직접 삭제하거나 브라우저 데이터를 초기화할 때까지 보유됩니다. 외부로 자동 전송·동기화되지 않습니다.

3. 익명 사용 통계 (선택, 활성화 시에만)

운영자가 서비스 개선을 위해 Plausible Analytics 를 활성화한 경우, 다음과 같은 익명·비식별 통계만 수집됩니다:

  • 페이지 방문 수, 익명 방문자 수 (개인 식별 불가)
  • 주요 기능 사용 이벤트 (예: 일일 챌린지 완료 — 개인 식별 정보 없음)

Plausible 은 쿠키를 사용하지 않으며, IP·기기 지문 등 개인정보를 저장하지 않습니다 (GDPR·PIPA 준수). 현재 베타 버전에서는 기본 비활성화 상태입니다.

4. 광고 (모바일 앱 버전, 향후)

웹(PWA)·데스크톱 버전에는 광고가 없습니다. 향후 iOS/Android 네이티브 앱으로 배포 시 Google AdMob 광고가 포함될 수 있으며, 이 경우 AdMob 의 개인정보 처리방침이 추가 적용됩니다. 현재 버전에는 광고가 포함되어 있지 않습니다.

5. 정보 보호 조치

  • 네트워크 통신 최소화 — PWA cache-first / 로컬 실행
  • CSP script-src 'self' — 외부 스크립트 차단 (분석 활성화 시 plausible.io 만 예외)
  • Storage 스키마 검증 — 변조 시 안전 복구

6. 이용자 권리

설정 → 데이터 백업으로 본인 데이터를 JSON 으로 내보낼 수 있습니다. 설정 → 전체 데이터 초기화로 모든 데이터를 영구 삭제할 수 있습니다.

7. 아동 개인정보

본 앱은 만 14세 이상 간호학과 학생·간호사 대상 교육 도구입니다. 만 14세 미만의 개인정보를 의도적으로 수집하지 않습니다.

8. 의료 면책

본 앱의 모든 컨텐츠는 국가시험 학습·교육 목적이며, 실제 임상 진단·치료·의료 행위의 근거로 사용할 수 없습니다. 정식 RN/MD 감수 전 베타 버전이며, 실제 임상 판단은 최신 가이드라인과 면허 의료인을 따라야 합니다.
