## 🟢 PART 1 — 휴대폰에서 앱 보기 (약 10분)

### 1단계. GitHub Pages 켜기 (앱을 인터넷에 올리기)

> "GitHub Pages" = 내 코드를 무료로 웹사이트로 만들어주는 기능입니다. 돈 안 듭니다.

1. 컴퓨터(또는 폰) 웹브라우저에서 이 주소로 가세요:
   **https://github.com/luiseluise0619-wq/nursing-simulation**

2. 로그인 안 되어 있으면 본인 GitHub 계정으로 로그인하세요.

3. 화면 위쪽 메뉴에서 **`Settings`** (톱니바퀴 ⚙️ 모양, "설정") 를 클릭하세요.
   - 안 보이면 `...` (점 세 개) 를 눌러보세요.

4. 왼쪽 메뉴를 아래로 내리다 보면 **`Pages`** 가 있습니다. 클릭하세요.

5. 화면 가운데 **"Build and deployment"** 라는 부분이 보입니다.
   - **Source** 칸: `Deploy from a branch` 를 선택하세요 (보통 이미 선택되어 있음).

6. 그 아래 **Branch** 칸에 드롭다운(▼) 두 개가 있습니다:
   - 첫 번째 드롭다운(▼): **`claude/app-rating-branch-report-ESRfZ`** 를 선택하세요.
     - 목록에 안 보이면 그냥 `main` 을 선택해도 됩니다 (단, 그러면 먼저 코드를 main 에 합쳐야 함 — 아래 PART 3 참고).
   - 두 번째 드롭다운(▼): **`/ (root)`** 를 선택하세요.

7. 옆에 있는 **`Save`** (저장) 버튼을 누르세요.

8. **1~2분 기다리세요.** (커피 한 잔 ☕)

9. 페이지를 새로고침(F5)하면 위쪽에 초록색으로:
   **"Your site is live at https://luiseluise0619-wq.github.io/nursing-simulation/"**
   같은 글이 뜹니다. 이 주소가 내 앱 주소입니다! 📍

### 3단계. 홈 화면에 앱처럼 설치하기 (선택)

> 이렇게 하면 진짜 앱처럼 아이콘이 생깁니다.

**아이폰 (Safari)**:
1. 아래쪽 **공유 버튼** (□ 안에 ↑ 화살표) 누르기
2. 아래로 내려서 **"홈 화면에 추가"** 누르기
3. **"추가"** 누르기 → 홈 화면에 아이콘 생김 ✅

**안드로이드 (Chrome)**:
1. 오른쪽 위 **점 세 개 (⋮)** 누르기
2. **"앱 설치"** 또는 **"홈 화면에 추가"** 누르기
3. **"설치"** 누르기 → 앱 서랍에 아이콘 생김 ✅

## 🟡 PART 2 — 앱 테스트 (약 30분, 깨진 곳 찾기)

> 아래 순서대로 눌러보고, **이상하거나 멈추거나 깨진 화면**이 있으면 메모하세요.

### 테스트 체크리스트 (하나씩 ✅)

- [ ] **첫 화면**: 약관 동의 화면이 뜨는가? 동의 체크 + 시작 눌러보기
- [ ] **온보딩**: 5장 안내 그림이 넘어가는가?
- [ ] **메인 메뉴**: 홈/학습/내 기록 3개 탭이 보이는가?
- [ ] **일일 챌린지**: 홈 → 일일 챌린지 → 10문제 끝까지 풀기 → 결과 화면 뜨는가?
- [ ] **듀티 시뮬레이션**: 홈 → 듀티 시뮬레이션 → 에피소드 끝까지 → 커리어 엔딩(승진/이직 등) 뜨는가?
- [ ] **커리어 스토리**: 학습 → 에피소드 → 📖 커리어 스토리 → 1막 인트로 소설처럼 뜨는가?
- [ ] **트레이닝**: 학습 → 트레이닝 → 🎲 8과목 통합 랜덤 → 10문제 후 세트 요약 뜨는가?
- [ ] **이미지 문제**: 문제 풀다 보면 ECG·욕창·체위 같은 그림이 나오는가? (가끔 나옴, 정상)
- [ ] **북마크**: 문제 화면 오른쪽 위 ⭐ 눌러서 저장 → 내 기록 → 북마크에 보이는가?
- [ ] **오답노트**: 일부러 틀린 후 → 내 기록 → 오답노트에 쌓이는가?
- [ ] **다크모드**: 위쪽 🌙 버튼 눌러서 어두운 테마로 바뀌는가?
- [ ] **민감 라벨**: 에피소드 목록에서 ⚠️ (자해·약물 등) 라벨이 일부 에피소드에 보이는가?
- [ ] **피드백**: 내 기록 → 앱 정보 → 📝 피드백 보내기 누르면 GitHub 열리는가?
- [ ] **소리**: 정답/오답 시 소리 나는가? 🔊 버튼으로 끄고 켜지는가?

### 깨진 곳 발견하면?
